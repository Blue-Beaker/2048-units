# 2048 Units
### A 2048-like game based on Physical Units.
### 一个基于物理单位的类2048游戏。

### Downloads/下载  

[2048units.html](2048units.html)

### Controls/控制  
移动：↑←↓→/WASD/拖拽  
撤销：UNDO按钮或鼠标右键  
Movement：↑←↓→/WASD/Swipe  
Undo: UNDO button or right mouse button
